## 🚀 Getting Started

### Prerequisites
- Python 3.9+
- Node.js 18+
- UNSW-NB15 dataset
- Root/sudo access (for packet capture)

### Quick Start
```bash
# Clone repository
git clone https://github.com/yourusername/smartshield.git
cd smartshield

# Setup Python
cd python && python -m venv venv && source venv/bin/activate
pip install -r requirements.txt

# Train models
python train_models.py --train-all

# Start system
cd .. && ./start.sh

# Open dashboard
# http://localhost:3000
```

### First Steps
1. ✅ Read README.md for overview

3. ✅ Train ML models on UNSW-NB15

5. ✅ Simulate attack with `attack_simulator.py`
6. ✅ Monitor dashboard for alerts
